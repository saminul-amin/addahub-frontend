# AddaHub - Community Event Management Platform

AddaHub is a modern, full-stack event management application designed to connect people through local activities. Whether you want to host a hiking trip, join a coding workshop, or find a yoga class nearby, AddaHub makes it seamless.

Built with the latest **Next.js 16** and **Express 5**, it features role-based access control, real-time updates, secure payments, and a highly responsive UI.

---

## Key Features

### For Users
- **Discover Events**: Advanced search and filtering by category (Social, Sports, Tech, etc.), date, and location.
- **Join & Participate**: Seamless booking flow with instant confirmation.
- **User Dashboard**: Track upcoming events and manage your profile.
- **Social Features**: View host profiles, leave reviews (coming soon), and see who else is attending.

### For Hosts
- **Event Creation**: Intuitive multi-step form with image uploads (Cloudinary).
- **Management Tools**: Edit event details, track participant lists, and manage capacity.
- **Monetization**: Set entry fees and receive payments securely (Stripe Integration).

### Security & Architecture
- **Role-Based Auth**: Distinct capabilities for Users, Hosts, and Admins.
- **Secure Authentication**: JWT-based stateless authentication with creating secure HTTP-only cookies.
- **Optimized Performance**: Next.js Server Components for SEO and fast initial loads, with Client Components for interactivity.

---

## Technology Stack

### Frontend
- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix Primitives)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State/Forms**: React Hook Form, Zod, Sonner (Toasts)

### Backend
- **Runtime**: [Node.js](https://nodejs.org/)
- **Framework**: [Express.js 5](https://expressjs.com/)
- **Database**: [MongoDB](https://www.mongodb.com/) (with Mongoose)
- **Validation**: Zod
- **Authentication**: JWT & Bcrypt
- **File Storage**: Cloudinary
- **Payments**: Stripe
